# **App Name**: ProfolioCraft

## Core Features:

- Project Showcase: Display previous projects with descriptions, images, and links.
- Skills Section: Highlight technical and soft skills with visual indicators of proficiency. Each skill should have a brief explanation.
- About Me Section: Introduce yourself with a professional bio, headshot, and contact information.
- AI-Powered Content Generation: Use a tool to suggest improvements for project descriptions or craft a compelling 'About Me' summary based on provided keywords and experience.
- Contact Form: Enable potential clients or employers to directly message you through the portfolio website.
- Responsive Design: Ensure the portfolio is accessible and visually appealing on all devices (desktop, tablet, mobile).

## Style Guidelines:

- Primary color: Dark blue (#34495E) to convey professionalism and stability.
- Background color: Light gray (#F0F3F4) for a clean and modern look.
- Accent color: Teal (#008080) for highlights and call-to-action buttons to provide a touch of vibrancy and visual interest.
- Body and headline font: 'Inter', a sans-serif font for a clean, modern and readable experience.
- Use a consistent set of minimalist icons to represent skills and categories.
- Implement a clean, grid-based layout with plenty of white space to improve readability and visual hierarchy.
- Use subtle transitions and animations to enhance user experience without being distracting (e.g., fade-in effects, hover animations).
